## 📚 **¿QUÉ ES LA DNDA?**

La **Dirección Nacional de Derecho de Autor (DNDA)** es la entidad colombiana encargada de registrar y proteger obras como software, libros, música, etc.

**Beneficios del registro:**
- ✅ Protección legal de tu software
- ✅ Prueba de autoría y fecha de creación
- ✅ Facilita demandas por plagio o uso no autorizado
- ✅ Aumenta el valor comercial del software
- ✅ Requisito para algunos contratos y licitaciones

### **PASO 2: Registro en Línea**

1. **Ir al sitio web de la DNDA:**
   - URL: https://www.derechodeautor.gov.co
   - Clic en "Registro de Obras"

2. **Crear cuenta (si no tienes):**
   - Registrarte con tu correo electrónico
   - Verificar correo
   - Completar perfil

3. **Iniciar solicitud:**
   - Clic en "Nueva Solicitud"
   - Seleccionar "Programa de Computador (Software)"
   - Llenar formulario en línea

4. **Cargar documentos:**
   - Subir código fuente (PDF)
   - Subir manual de usuario (PDF)
   - Subir cédula (PDF)

5. **Pagar derechos:**
   - Generar recibo de pago
   - Pagar en banco autorizado o PSE
   - Cargar comprobante

6. **Enviar solicitud:**
   - Revisar toda la información
   - Enviar solicitud
   - Guardar número de radicado

---

### **PASO 3: Seguimiento**

1. **Tiempo de respuesta:** 15-30 días hábiles
2. **Consultar estado:** En el sitio web con tu número de radicado
3. **Correcciones:** Si hay observaciones, tienes 30 días para responder
4. **Aprobación:** Recibirás certificado por correo

## ⚠️ **CONSIDERACIONES IMPORTANTES**

### **1. Autoría**
- Si trabajaste con otras personas, debes listarlas como coautores
- Si usaste código de terceros (librerías), debes mencionarlo
- Las librerías open-source no afectan el registro

### **2. Código Fuente**
- **NO incluyas contraseñas o API keys** en el código que envías
- Reemplaza información sensible con `[REDACTED]`
- El código debe ser legible y tener comentarios

### **3. Confidencialidad**
- La DNDA mantiene confidencial el código fuente
- Solo se hace público el nombre y titular de la obra
- El código NO se publica en internet

### **4. Vigencia**
- La protección dura **toda tu vida + 80 años**
- No necesitas renovar el registro
- Puedes registrar nuevas versiones

# 3D Models Folder

## Quick Download Instructions

To see 3D models in the geometry visualizer, download `.glb` files and place them here.

### Download a Test Cube (2 minutes):

1. **Open this link in your browser:**
   https://github.com/KhronosGroup/glTF-Sample-Models/blob/master/2.0/Box/glTF-Binary/Box.glb

2. **Click the "Download" button** (top right of the page)

3. **Move the downloaded file** to this folder

4. **Rename it** from `Box.glb` to `cube.glb`

5. **Refresh your browser** - you'll see a 3D cube!

---

## Files Needed:

Place these `.glb` files in this folder:

- `cube.glb` - For the cube shape
- `sphere.glb` - For the sphere shape  
- `pyramid.glb` - For the pyramid shape
- `cylinder.glb` - For the cylinder shape
- `cone.glb` - For the cone shape
- `rectangular-prism.glb` - For the rectangular prism shape

---

## Where to Get Models:

### Free Sources:
- **Sketchfab**: https://sketchfab.com (filter: Downloadable + Free)
- **NASA 3D**: https://nasa3d.arc.nasa.gov (space models)
- **glTF Samples**: https://github.com/KhronosGroup/glTF-Sample-Models

### AI Generation:
- **Meshy.ai**: https://meshy.ai (free tier: 200 credits/month)
- **Luma AI**: https://lumalabs.ai

---

## File Format:

- **Must be `.glb` format** (not .obj, .fbx, .stl, etc.)
- **Keep files under 5MB** for fast loading
- **Use low-poly models** for better performance

---

**Once you add files here, refresh the browser and click on shapes to see them in 3D!**
